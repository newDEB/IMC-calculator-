# IMC Calculator 🧮

Mon tout premier projet en Python ! Un script simple et interactif pour calculer l'Indice de Masse Corporelle (IMC).

## 🚀 Fonctionnalités
- Demande le poids en kilogrammes.
- Demande la taille en mètres (ex: 1.75).
- Calcule et affiche l'IMC instantanément.

## 💻 Comment le lancer
1. Télécharge le fichier `IMC.py`.
2. Lance-le dans ton terminal avec la commande :
   ```bash
   python IMC.py
